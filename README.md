##

### Pre-requisites

* Cocopods
* node.js 
* npm

### Installation

From source directory:

~~~
npm install
~~~

Then:

~~~
pod install
~~~

### Run code

From source directory:

~~~
open Game1.xcworkspace
~~~

Click the run button in Xcode.
